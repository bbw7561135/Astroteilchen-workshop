This is a prgram to plot gamma-ray spectra generated by protons and electrons via proton-proton interactions or by inverse compton, bremsstrahlungs or synchrotron emission.

To run the prgrogram you need:

Python 2.7.x with x >9
Scipy

Your Scipy should include numpy and matplotlib

once you checked out the repository you chould make the main script executable:

chmod +x plot_spectrum.py

you can run the program for example like this:

./plot_spectrum.py --spec sbpl --par "49. 1. -2. 6. 0.1" --proc ic --erange 1e6 1e12

alternatively in case you do not want to execute the script by itself you can write:

python plot_spectrum.py --spec sbpl --par "49. 1. -2. 6. 0.1" --proc ic --erange 1e6 1e12

to see all options type:

./plot_spectrum.py --h
